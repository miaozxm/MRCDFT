# 实验笔记使用说明

## 文件位置
main/experiment_note.txt

## 工作流程

### 方式1：使用默认笔记（快速运行）
1. 不创建 experiment_note.txt 文件
2. 直接运行 main.sh
3. 系统会自动生成包含基本信息的默认笔记

### 方式2：自定义笔记（推荐）
1. 在 main/ 目录下创建 experiment_note.txt
2. 填写实验目的、相关实验等信息
3. 运行 main.sh
4. 系统会复制你的自定义笔记到实验目录

## 笔记模板

```
# 实验记录
# ============================================

## 实验目的
研究 Ca40 在不同形变参数下的能量曲线

## 相关实验
- 参考实验：40Ca_8_2026-06-01-10-00-00（Nf=8 基准计算）
- 对比实验：40Ca_10_2026-06-05-14-30-00（不同约束条件）

## 备注
本次测试使用 beta2 从 -0.5 到 0.5，步长 0.1
```

## 查找同系列实验

```bash
# 查找所有 Ca40 相关实验
find ../Labs -name ".group_id" -exec grep -l "40Ca_study" {} \;

# 查看某个实验的笔记
cat ../Labs/40Ca/40Ca_10_2026-06-08-16-59-42/experiment_note.txt
```

## 注意事项
- 如果 experiment_note.txt 存在但为空，也会生成默认笔记
- 每次运行都会创建新的实验目录，不会覆盖历史笔记
- .group_id 文件用于实验关联，无需手动编辑
- **脚本运行结束后会自动清空 experiment_note.txt**，下次运行时将生成新的默认笔记
- 如需保留自定义笔记模板，请在运行前备份或运行后立即重新编辑
